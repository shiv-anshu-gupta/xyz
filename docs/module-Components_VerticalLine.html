    Components/VerticalLine

    Manages keyboard shortcuts for adding, removing, and navigating vertical line markers on COMTRADE charts. Vertical lines are used to mark specific time points for measurement and delta calculations.

    Keyboard Shortcuts:

    • Alt + 0: Clear all vertical lines
    • Alt + 1: Add vertical line at current cursor position
    • Alt + 2: Go to previous vertical line
    • Alt + 3: Go to next vertical line
    • Alt + 4: Delete current vertical line

    Features:

    • Synchronized vertical lines across all charts
    • Automatic delta calculation (time and value differences)
    • Display results in fixed overlay
    • Cursor navigation to markers

    Description:

    • Vertical Line Position Control

      A UI control panel component that provides a range slider and numeric input for precisely positioning vertical measurement lines across all charts. Enables users to select specific time points for detailed waveform analysis.

      Design Philosophy

      PrincipleDescription
      Dual InputBoth slider and numeric field for coarse/fine control
      SynchronizedSlider and input always show same value
      Callback-BasedExternal handlers for slider and input changes
      Configurable RangeMin/max/step values from COMTRADE time range

      Key Features

      • Range Slider — Drag to quickly navigate time range
      • Numeric Input — Type exact time values with precision
      • Step Control — Configurable step increment (default 0.001)
      • Value Clamping — Enforces min/max boundaries
      • Live Updates — Callbacks fire on every value change
      • Programmatic API — setValue() for external control

      Control Panel Layout

      ┌────────────────────────────────────┐
│ Vertical Line Position             │
│ [═══════●══════════════] [0.500  ] │
│  min                max    input   │
└────────────────────────────────────┘
    See:
    • module:components/initVerticalLineControl - Initialization wrapper
    • module:components/handleVerticalLineShortcuts - Keyboard shortcuts
    Example
    const control = createVerticalLineControl({
  containerId: 'control-panel',
  minValue: 0,
  maxValue: 1.5,        // From COMTRADE duration
  step: 0.001,
  initialValue: 0.75,
  onSliderChange: (value) => {
    updateAllCharts(value);
  },
  onInputChange: (value) => {
    updateAllCharts(value);
  }
});

// Programmatic update
control.setValue(0.5);

document.addEventListener('keydown', (e) => {
