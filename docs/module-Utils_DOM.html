    Utils/DOM

    RAF-based DOM update queuing system for batching rapid DOM updates. Prevents layout thrashing by grouping updates and executing them on requestAnimationFrame.

    Features:

    • Batches DOM updates to single RAF frame
    • Deduplicates updates to same element/property
    • Memory-safe: Automatically cleans up after execution
    • No memory leaks: Proper RAF cancellation on cleanup
    • Supports multiple queues for different update types
    • Lightweight and modular

    Performance Impact:

    • Without selectiveUpdate: N updates = N DOM reflows
    • With selectiveUpdate: N updates = 1 reflow per RAF frame
    • Typical speedup: 30x-100x for rapid updates

  dedupeKey: 'el-textContent'
});

// All updates execute on next RAF frame
// Later updates with same dedupeKey replace earlier ones

    • sidebarResize module

    Description:

    • Global Sidebar State Manager

      Manages visibility state of all sidebars/drawers to ensure only one is visible at a time. Provides a centralized API for registering, showing, and hiding sidebar components with mutual exclusion enforcement.

      Design Philosophy

      PrincipleDescription
      Single ActiveOnly one sidebar can be visible at any time
      Registry PatternSidebars register themselves with show/hide/isOpen methods
      Auto-CloseOpening one sidebar automatically closes any other open sidebar
      Escape HandlingGlobal escape key listener closes active sidebar

      Key Features

      • Registration — Sidebars register with show/hide/isOpen callbacks
      • Mutual Exclusion — Automatically hides other sidebars when one opens
      • Query API — Check if any sidebar is open, get active sidebar ID
      • Escape Key — Global keyboard handler for closing sidebars
      • Debug Support — getRegisteredSidebars() for troubleshooting
    See:
    • module:components/DeltaDrawer - Delta values sidebar
    • module:components/AnalysisSidebar - Phasor analysis sidebar
    Example
    import { sidebarStore } from './sidebarStore.js';

// Register a sidebar
sidebarStore.register('delta-drawer', {
  show: () => drawer.show(),
  hide: () => drawer.hide(),
  isOpen: () => drawer.isOpen()
});

// Show a sidebar (auto-hides others)
sidebarStore.show('delta-drawer');

// Check active sidebar
if (sidebarStore.getActiveSidebar() === 'delta-drawer') {
  console.log('Delta drawer is active');
}

    sidebarResize module

    Description:

    • RAF-based DOM update queuing system for batching rapid DOM updates. Prevents layout thrashing by grouping updates and executing them on requestAnimationFrame.

      Features:

      • Batches DOM updates to single RAF frame
      • Deduplicates updates to same element/property
      • Memory-safe: Automatically cleans up after execution
      • No memory leaks: Proper RAF cancellation on cleanup
      • Supports multiple queues for different update types
      • Lightweight and modular

      Performance Impact:

      • Without selectiveUpdate: N updates = N DOM reflows
      • With selectiveUpdate: N updates = 1 reflow per RAF frame
      • Typical speedup: 30x-100x for rapid updates
    Example
    import { createDOMUpdateQueue } from './utils/domUpdateQueue.js';

const queue = createDOMUpdateQueue();

// Queue updates
queue.queueUpdate({
  element: el,
  updateFn: () => { el.textContent = 'Updated'; },
  dedupeKey: 'el-textContent'
});

// All updates execute on next RAF frame
// Later updates with same dedupeKey replace earlier ones
