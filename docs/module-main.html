# inner deleteChannelByID(channelID) → {boolean}

Removes a channel from all related arrays in channelState, dataState, and the raw data object. This ensures all arrays remain synchronized with the same length. The operation is immediate and irreversible. After deletion, reactive subscribers trigger and charts are automatically recreated with the updated series list.
