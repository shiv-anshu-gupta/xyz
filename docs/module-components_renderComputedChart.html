Module

components/renderComputedChart

Computed Channel Chart Renderer

Orchestrates rendering of computed channels by delegating to shared renderers.

What This Module Does

StepAction
1Reads computed channels from cfg.computedChannels + data.computedData
2Separates analog vs digital via madeFrom field
3Groups channels by their group field
4Calls renderSingleAnalogChart() for analog types
5Calls renderSingleDigitalChart() for digital types

Design Philosophy

  • Unified Rendering — Uses same renderers as COMTRADE data
  • Computed-Only Mode — Passes channelIndices=[] to signal computed-only
  • Group-Based — Each channel's group field determines chart ownership
  • Type-Aware — Routes to correct renderer based on madeFrom

Ownership Rule

Computed channels with group matching an existing analog/digital group are rendered in that group's chart. Unmatched groups get standalone charts.

Data Flow

  1. cfg.computedChannels[i] + data.computedData[i]
  2. ↓ buildComputedChannelObjects()
  3. ↓ separateByType() + groupByGroup()
  4. ↓ analogGroups / digitalGroups
  5. ↓ renderSingleAnalogChart() / renderSingleDigitalChart()
  6. Charts with computed channels only
graph TD A[cfg.computedChannels + data.computedData] --> B[buildComputedChannelObjects] B --> C[Combined channel objects] C --> D{madeFrom field} D -->|analog| E[groupByGroup - analog] D -->|digital| F[groupByGroup - digital] D -->|null/unknown| G[Determine from formula] E --> H[renderSingleAnalogChart per group] F --> I[renderSingleDigitalChart per group] H --> J[Analog charts] I --> K[Digital charts]

  • module:components/renderSingleDigitalChart

Methods

# static renderComputedChart(cfg, data, chartsContainer, charts, verticalLinesX, channelState) → {void}

Render computed channels using the shared single-chart renderers. **Execution Flow:** 1. Build combined channel objects from cfg + data 2. Clean up any old computed charts 3. Separate channels by type (madeFrom: "analog" vs "digital") 4. Group each type by their `group` field 5. Call appropriate renderer for each group
Parameters:
Name Type Description
cfg Object Configuration with computedChannels metadata array
computedChannels Array.<Object> Computed channel definitions
computedChannels[].id string Channel identifier
computedChannels[].name string Display name
computedChannels[].group string Target group ID
computedChannels[].madeFrom string Source type ("analog"|"digital"|null)
computedChannels[].color string Hex color
computedChannels[].unit string Unit string
data Object Data object with computedData values array
computedData Array.<Array> 2D array of computed values
time Array.<number> Time array (used for X-axis)
chartsContainer HTMLElement Container for chart elements
charts Array.<uPlot> Array storing all chart instances
verticalLinesX Object Reactive array of vertical line positions
channelState Object Reactive state for channel customizations
