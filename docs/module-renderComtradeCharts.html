Module

renderComtradeCharts

Main chart rendering orchestrator for COMTRADE application. Handles the complete workflow of rendering analog and digital charts from parsed COMTRADE data, with support for drag-and-drop reordering and vertical line overlays.

# inner renderComtradeCharts(cfg, data, chartsContainer, charts, verticalLinesX, createState, calculateDeltas, TIME_UNIT, channelState) → {void}

Render COMTRADE charts (analog and digital) in the container. Orchestrates the complete chart rendering workflow: 1. Clears previous charts and DOM 2. Sets up drag-and-drop functionality 3. Renders analog charts with smart grouping 4. Renders digital charts if present 5. Calculates and displays time/value deltas for vertical lines
Parameters:
Name Type Description
cfg Object COMTRADE configuration object
analogChannels Array Array of analog channel definitions
digitalChannels Array Array of digital channel definitions
samplingRate number Sampling rate in samples/second
data Object Parsed COMTRADE data
time Float64Array Time array (uniform spacing)
analogData Array.<Float64Array> 2D array [channel][sample]
digitalData Array.<Float64Array> 2D array [channel][sample]
chartsContainer HTMLElement DOM element to render charts into
charts Array Array reference to store uPlot instances
verticalLinesX Array | State Array of vertical line X positions (optional, can be createState)
createState function Reactive state creator function
calculateDeltas function Function to calculate deltas at vertical lines
TIME_UNIT string Time unit label ('seconds', 'milliseconds', etc)
channelState Object Reactive state for channel metadata (colors, labels, scales, units, inverts)
