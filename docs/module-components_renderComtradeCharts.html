Module

components/renderComtradeCharts

Main Chart Rendering Orchestrator

Entry point for chart visualization after COMTRADE file parsing. Orchestrates rendering of all chart types: analog, digital, and computed.

Responsibilities

TaskDescription
CleanupClear existing charts and metadata
Drag & DropEnable chart reordering
CoordinationRender analog → digital → computed
SubscriptionsManage chart metadata updates
Delta CalcTrigger calculations for vertical markers

Rendering Pipeline

  1. destroyCharts() + clearContainer()
  2. analyzeGroupsAndPublishMaxYAxes()
  3. Phase 1: Analog Charts (one per group)
    ↳ Include computed channels with matching group
  4. Phase 2: Digital Charts (one per group)
  5. Phase 3: Standalone Computed Channels
  6. Setup delta calculations

Ownership Rule

Computed channels with group matching an analog group render inside that chart. Only unmatched groups get standalone charts.
graph TD A[renderComtradeCharts] --> B[clearAllCharts] B --> C[destroyCharts] C --> D[setupChartDragAndDrop] D --> E[analyzeGroupsAndPublishMaxYAxes] E --> F[Phase 1: Analog Charts] F --> G[For each analog group] G --> H[renderSingleAnalogChart] H --> I[Phase 2: Digital Charts] I --> J[For each digital group] J --> K[renderSingleDigitalChart] K --> L[Phase 3: Computed Charts] L --> M[renderComputedChart] M --> N[Setup delta calculations]

Methods

# static renderComtradeCharts(cfg, data, chartsContainer, charts, verticalLinesX, createState, calculateDeltas, TIME_UNIT, channelState) → {void}

Render all COMTRADE charts in the container. This is the main orchestration function that coordinates rendering of analog, digital, and computed channel charts. **Execution Flow:** ``` 1. Clear existing charts and metadata 2. Setup drag-and-drop 3. Analyze groups for Y-axis alignment 4. Render analog charts (one per group) - Include computed channels that belong to each group 5. Render digital charts (one per group) 6. Render standalone computed channels 7. Calculate deltas for vertical markers ``` **Memory Management:** - Properly destroys old chart instances before creating new ones - Disconnects ResizeObservers and event listeners - Clears chart metadata store

const cfg = parseCFG(cfgText);
const data = parseDAT(datBuffer, cfg);

renderComtradeCharts(

  charts,

  createState,
  calculateDeltas,
  "ms",
  channelState

renderComtradeCharts(cfg, data, container, charts, ...);

charts.forEach(chart => {
  console.log(chart._userGroupId, chart._chartType);
