    uPlot Plugin: Auto Unit & SI Prefix Scaling

    Automatically applies the best SI prefix (p, n, µ, m, k, M, G, T) to axis labels and tick values based on the visible data range. Supports both X and Y axes with user-configurable scale factors for unit conversion.

    Design Philosophy

    PrincipleDescription
    Auto-ScalingDynamically selects SI prefix based on visible data magnitude
    User OverrideaxesScales allows manual unit conversion (mV→V, ms→s)
    Label PreservationOriginal axis labels stored and restored with prefix updates
    Defensive CodingGracefully handles missing axis definitions and custom ticks

    Key Features

    • SI Prefix Auto-Selection — p, n, µ, m, (base), k, M, G, T
    • Multi-Axis Support — Works with X and multiple Y axes
    • User Scale Factors — Multiply raw values before prefix selection
    • Dynamic Updates — Recalculates on zoom/pan via setScale hook
    • Label Format — Expects "Label (Unit)" format, e.g., "Voltage (V)"

    User Scale Explanation

    The axesScales array provides multiplicative factors for each axis:

    • [1, 0.001, 1] — Y0 axis data is in mV, display as V
    • [0.001, 1, 1] — X axis data is in ms, display as s

    Limitations

    • Assumes axis labels contain units in parentheses
    • Does not support dynamic unit changes after chart creation
    • Overrides any existing axis.values function

    Description:

    • uPlot Plugin: Horizontal Zoom & Pan

      Enables keyboard and mouse-based horizontal zooming and panning across synchronized charts. Uses modifier keys (Shift, Alt) to distinguish zoom from pan operations.

      Design Philosophy

      PrincipleDescription
      Modular DesignAccepts getCharts function, no global state dependency
      Chart SynchronizationAll charts zoom/pan together via syncKey
      Modifier KeysShift for zoom, Alt for pan—prevents accidental operations
      Clean CleanupProper event handler removal on chart destroy

      Key Features

      • Shift+Scroll — Zoom in/out horizontally
      • Alt+Scroll — Pan left/right horizontally
      • Shift+Arrow — Zoom with keyboard
      • Alt+Arrow — Pan with keyboard
      • Mouse-Anchored Zoom — Zoom centers on cursor position
      • Sync Group — Charts with same syncKey update together

      Control Mapping

      InputAction
      Shift + Scroll UpZoom In (0.8x range)
      Shift + Scroll DownZoom Out (1.25x range)
      Alt + Scroll UpPan Left (10% of range)
      Alt + Scroll DownPan Right (10% of range)
      Shift + Left/RightZoom via keyboard
      Alt + Left/RightPan via keyboard
    See:
    • module:plugins/verticalLinePlugin - Vertical measurement lines
    Example
    import horizontalZoomPanPlugin from './plugins/horizontalZoomPanPlugin.js';

const opts = {
  plugins: [
    horizontalZoomPanPlugin('globalSync', () => window.charts)
  ]
};
    Description:

    • uPlot Plugin: Vertical Line & Crosshair Points

      Enables placing, dragging, and visualizing vertical measurement lines on uPlot charts. Lines are synchronized across all charts and display crosshair points where they intersect series data. Used for delta calculations and time-point analysis.

      Design Philosophy

      PrincipleDescription
      Reactive StateLines stored in verticalLinesXState, auto-syncs across charts
      Drag SupportLines can be dragged to new positions with mouse
      Event CaptureUses capture phase to prevent selection box during drag
      Debounced UpdatesDelta calculations debounced to prevent UI freezing

      Key Features

      • Add Lines — Alt+1 adds vertical line at cursor position
      • Drag Lines — Click and drag to reposition lines
      • Crosshair Points — Colored dots at series intersections
      • Line Colors — Configurable colors from crosshairColors palette
      • Chart Sync — All charts show same vertical lines
      • Delta Display — Triggers delta calculations on line change

      Keyboard Shortcuts

      ShortcutAction
      Alt+0Clear all vertical lines
      Alt+1Add vertical line at cursor
      Alt+2Go to previous line
      Alt+3Go to next line
      Alt+4Delete current line
    See:
    • module:components/handleVerticalLineShortcuts - Keyboard handler
    • module:utils/calculateDeltas - Delta calculation utilities
    Example
    import verticalLinePlugin from './plugins/verticalLinePlugin.js';

const opts = {
  plugins: [
    verticalLinePlugin(verticalLinesXState, getCharts, {
      lineColors: ['red', 'blue', 'green'],
      lineWidth: 2,
      pointRadius: 5
    })
  ]
};

    Methods

    (static) createDigitalFillPlugin(signals) → {Object}

    Description:

    • uPlot plugin for rendering filled regions under digital (boolean) signals.

      This plugin draws filled rectangles under digital signals (step lines) when the signal is in the "high" state (as defined by targetVal). It supports multiple digital signals, each with its own vertical offset, color, and target value.

      The fill is drawn in sync with zoom/pan and respects the current axis scales.

    Example
    import { createDigitalFillPlugin } from './src/plugins/digitalFillPlugin.js';

const xVals = [0, 1, 2, 3, 4, 5, 6];
const signal1 = [1, 0, 1, 1, 0, 1, 0];
const signal2 = [1, 1, 1, 0, 1, 1, 1];

const signals = [
  { signalIndex: 1, offset: 0, color: 'rgba(0, 150, 255, 0.3)', targetVal: 1 },
  { signalIndex: 2, offset: 3, color: 'rgba(255, 100, 100, 0.3)', targetVal: 1 }
];

const opts = {
  width: 600,
  height: 300,
  scales: { x: { time: false }, y: { min: 0, max: 6 } },
  axes: [{}, {}],
  series: [
    {}, // x axis
    { label: "Signal 1", stroke: "transparent" },
    { label: "Signal 2", stroke: "transparent" }
  ],
  plugins: [
    createDigitalFillPlugin(signals)
  ]
};

const data = [xVals, signal1, signal2];
new uPlot(opts, data, document.getElementById("chart-digital"));
    Parameters:
    Name Type Description
    signals Array.<Object>

    Array of signal configuration objects. Each object should have: - signalIndex {number}: Index of the signal in the uPlot data array (e.g., 1 for the first signal after x). - offset {number}: Vertical offset to apply to the signal (for stacking multiple signals). - color {string}: Fill color for the high region (CSS color, e.g., 'rgba(0,150,255,0.3)'). - targetVal {number}: Value to consider as "high" (usually 1).
    Returns:

    uPlot plugin object to be included in the plugins array of uPlot options.

    Type
    Object

    • uPlot Plugin: Auto Unit & SI Prefix Scaling

      Automatically applies the best SI prefix (p, n, µ, m, k, M, G, T) to axis labels and tick values based on the visible data range. Supports both X and Y axes with user-configurable scale factors for unit conversion.

      Design Philosophy

      PrincipleDescription
      Auto-ScalingDynamically selects SI prefix based on visible data magnitude
      User OverrideaxesScales allows manual unit conversion (mV→V, ms→s)
      Label PreservationOriginal axis labels stored and restored with prefix updates
      Defensive CodingGracefully handles missing axis definitions and custom ticks

      Key Features

      • SI Prefix Auto-Selection — p, n, µ, m, (base), k, M, G, T
      • Multi-Axis Support — Works with X and multiple Y axes
      • User Scale Factors — Multiply raw values before prefix selection
      • Dynamic Updates — Recalculates on zoom/pan via setScale hook
      • Label Format — Expects "Label (Unit)" format, e.g., "Voltage (V)"

      User Scale Explanation

      The axesScales array provides multiplicative factors for each axis:

      • [1, 0.001, 1] — Y0 axis data is in mV, display as V
      • [0.001, 1, 1] — X axis data is in ms, display as s

      Limitations

      • Assumes axis labels contain units in parentheses
      • Does not support dynamic unit changes after chart creation
      • Overrides any existing axis.values function
    See:
    • module:utils/scaleUtils - SI prefix calculation utilities
    • module:utils/constants - SI_UNITS table
    Example
    import autoUnitScalePlugin from './plugins/autoUnitScalePlugin.js';

const opts = {
  ...otherUplotOptions,
  plugins: [autoUnitScalePlugin({ axesScales: [1, 0.001, 1] })]
  // Y0 axis: data in mV, displayed as V with appropriate SI prefix
};
new uPlot(opts, data, target);

    (static) createDigitalFillPlugin(signals) → {Object}

    Description:

    • uPlot plugin for rendering filled regions under digital (boolean) signals.

      This plugin draws filled rectangles under digital signals (step lines) when the signal is in the "high" state (as defined by targetVal). It supports multiple digital signals, each with its own vertical offset, color, and target value.

      The fill is drawn in sync with zoom/pan and respects the current axis scales.

    Example
    import { createDigitalFillPlugin } from './src/plugins/digitalFillPlugin.js';

const xVals = [0, 1, 2, 3, 4, 5, 6];
const signal1 = [1, 0, 1, 1, 0, 1, 0];
const signal2 = [1, 1, 1, 0, 1, 1, 1];

const signals = [
  { signalIndex: 1, offset: 0, color: 'rgba(0, 150, 255, 0.3)', targetVal: 1 },
  { signalIndex: 2, offset: 3, color: 'rgba(255, 100, 100, 0.3)', targetVal: 1 }
];

const opts = {
  width: 600,
  height: 300,
  scales: { x: { time: false }, y: { min: 0, max: 6 } },
  axes: [{}, {}],
  series: [
    {}, // x axis
    { label: "Signal 1", stroke: "transparent" },
    { label: "Signal 2", stroke: "transparent" }
  ],
  plugins: [
    createDigitalFillPlugin(signals)
  ]
};

const data = [xVals, signal1, signal2];
new uPlot(opts, data, document.getElementById("chart-digital"));

    Array of signal configuration objects.
