Module

renderAnalogCharts

Renders analog channel charts from COMTRADE data with intelligent grouping, drag-and-drop support, and interactive features (tooltips, vertical lines, deltas). Features: - Smart grouping of analog channels (user-defined or automatic) - One uPlot chart per group with multiple Y-axes - Drag bars between charts for reordering - Interactive tooltips showing values on hover - Vertical line overlays for time markers - Delta calculations (time and value differences) - Channel inversion (negative values) - Configurable scales and units per channel

# inner renderAnalogCharts(cfg, data, chartsContainer, charts, verticalLinesX, channelState, autoGroupChannels) → {void}

Render analog charts with smart grouping and drag-and-drop support. Groups analog channels intelligently (by phase, unit, or user preference) and creates one uPlot chart per group. Each chart displays multiple analog series with separate Y-axes, interactive tooltips, and overlays.
Parameters:
Name Type Description
cfg Object COMTRADE config
analogChannels Array Array of {ph, ccname, uu, a, b} objects
data Object Parsed COMTRADE data
time Float64Array Time array (samples)
analogData Array.<Float64Array> 2D array [channel][sample]
chartsContainer HTMLElement DOM element for charts
charts Array Array to store uPlot instances [chart1, chart2, ...]
verticalLinesX Array | State Vertical line X positions
channelState Object Channel metadata state
analog.groups Array Group names for each analog channel
analog.lineColors Array Colors for each series
analog.scales Array Scale factors for each series
analog.units Array Units for each series
analog.inverts Array Invert flags for each series
autoGroupChannels function Function to auto-group channels
