    ComputedChannelMetadata

    Utils/ComputedChannels~ ComputedChannelMetadata

    new ComputedChannelMetadata()

    Description:

    • Computed Channel Metadata Manager Class. Provides O(1) lookup by ID and maintains insertion order.

    Example
    const manager = new ComputedChannelMetadata();
manager.set("ch1", { name: "Channel 1", color: "#FF0000" });
console.log(manager.get("ch1").name); // "Channel 1"
