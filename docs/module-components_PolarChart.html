    Components/PolarChart

    Phasor Diagram Component

    An SVG-based polar/phasor chart that visualizes three-phase electrical quantities (currents and voltages) as vectors in polar coordinates. Displays magnitude and phase angle relationships for power system analysis.

    Design Philosophy

    PrincipleDescription
    SVG RenderingPure SVG implementation since uPlot lacks native polar support
    Auto-ScalingMagnitudes automatically scaled to fit 70% of chart radius
    Debounced Updates100ms debounce prevents UI freezing from rapid cursor movements
    Batched DOM OpsUses DocumentFragment for optimal render performance
    Graceful FallbackShows mock data if real COMTRADE data unavailable

    Key Features

    • Three-Phase Display — Shows IA/IB/IC currents and VA/VB/VC voltages
    • Vector Arrows — Magnitude represented by length, angle by direction
    • Grid Circles — Reference circles at 20%, 40%, 60%, 80%, 100% radius
    • Axis Labels — 0°, 90°, 180°, 270° degree markers
    • Color Coding — Distinct colors for each phase (Cyan/Blue/Green for I, Orange/Yellow/Red for V)
    • Dynamic Time Index — Update phasors at any sample point in the waveform
    • Phasor Limiting — Caps at 12 phasors to maintain performance

    Rendering Pipeline

    1. Clear container and calculate SVG dimensions
    2. Create DocumentFragment for batched operations
    3. Draw background circle and grid circles
    4. Draw axis lines and degree labels
    5. Calculate max magnitude for auto-scaling
    6. Draw phasor vectors with arrows and labels
    7. Append entire fragment to DOM in single operation
    8. Log render time for performance monitoring

    Phase Angle Convention

    PhaseAngleDescription
    A0°Reference phase
    B240°Lags A by 120°
    C120°Leads A by 120°

    • Canvas-based Phasor Diagram Renderer

      A high-performance HTML5 Canvas-based polar/phasor chart that replaces the SVG implementation for significantly faster rendering (10x+ improvement). Uses direct canvas drawing operations for optimal performance during rapid cursor movements.

      Design Philosophy

      PrincipleDescription
      Canvas over SVGDirect pixel manipulation for 10x faster rendering
      Immediate ModeNo DOM nodes created, just draw commands
      Auto-ScalingCanvas resizes to container dimensions
      Render GuardisRendering flag prevents overlapping draw calls

      Key Features

      • HTML5 Canvas — Direct 2D context drawing for speed
      • Three-Phase Display — Shows IA/IB/IC and VA/VB/VC phasors
      • Auto-Scale — Adapts to container size
      • Grid Drawing — Circular grid with degree markers
      • Arrow Heads — Proper vector arrows with angle calculation
      • Label Positioning — Automatic label placement near arrow tips

      Performance Comparison

      MethodTypical Render TimeDOM Nodes
      SVG (PolarChart.js)~50-100msMany (one per element)
      Canvas (this file)~5-10ms1 (canvas element)

    • module:components/PolarChart - SVG-based alternative
    • module:components/AnalysisSidebar - Host sidebar component

    // Initialize canvas-based polar chart
const polarChart = new PolarChartCanvas('polar-chart-container');
polarChart.init();

// Update with phasor data
polarChart.updateData([
  { label: 'IA', magnitude: 100, angle: 0, color: '#00d9ff' },
  { label: 'IB', magnitude: 95, angle: 240, color: '#2196f3' },
  { label: 'IC', magnitude: 98, angle: 120, color: '#10b981' }
]);
    Description:

    • Phasor Chart Cursor Integration

      Connects the polar/phasor chart to the main waveform charts via vertical line positions. When users place vertical measurement lines on waveforms, this module updates the phasor diagram to show the instantaneous phasor values at that specific time point.

      Design Philosophy

      PrincipleDescription
      Time SynchronizationPhasor updates match vertical line position
      Nearest IndexBinary search finds closest sample to cursor position
      Debounced UpdatesPrevents excessive phasor redraws during drag
      State SubscriptionListens to verticalLinesX reactive state changes

      Key Features

      • Automatic Updates — Phasor diagram updates on vertical line change
      • Time Index Lookup — Finds nearest sample index to cursor X position
      • Multiple Line Support — Works with any number of vertical markers
      • Cursor Tracking — Can also update from chart cursor movement
    See:
    • module:components/PolarChart - SVG phasor diagram
    • module:components/PolarChartCanvas - Canvas phasor diagram
    • module:components/AnalysisSidebar - Host sidebar
    Example
    // Setup integration after charts are rendered
setupPolarChartWithVerticalLines(
  polarChart,      // PolarChart instance
  cfg,             // COMTRADE config
  data,            // COMTRADE data
  verticalLinesX,  // Reactive state for vertical line positions
  charts           // Array of uPlot chart instances
);
