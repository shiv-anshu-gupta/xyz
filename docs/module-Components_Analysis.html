    Delta Values Sidebar Drawer

    A slide-out sidebar component that displays detailed crosshair/vertical line values in a searchable, formatted table. Shows measurement data at vertical marker positions placed on charts using Alt+1 keyboard shortcut.

    Design Philosophy

    PrincipleDescription
    Pre-built HTMLDOM structure defined in index.html, component handles behavior only
    CSS Variable LayoutUses --main-content-width and --sidebar-width for responsive resizing
    DeduplicationHash-based comparison prevents redundant table re-renders
    Lazy LoadingTable renderer created on-demand when drawer is opened
    Store IntegrationRegistered with sidebarStore for single-sidebar-at-a-time behavior

    Key Features

    • Vertical Line Values — Displays channel values at each vertical marker position
    • Delta Calculations — Shows differences between consecutive vertical lines
    • Time Headers — Dynamic column headers showing actual timestamp values (μs)
    • Search Filtering — Real-time search to filter table rows by channel name
    • Resizable Width — Percentage-based width (15-70%) controlled via CSS variables
    • Empty State — Helpful message when no vertical lines are placed

    Data Flow Pipeline

    1. User places vertical lines on chart (Alt+1)
    2. verticalLinesX state updates with new positions
    3. Parent calls deltaDrawer.update(deltaData, lineCount)
    4. Hash comparison checks if data changed
    5. formatTableData() transforms sections into row format
    6. DeltaTableRenderer creates/updates HTML table
    7. Search input filters visible rows in real-time

    DOM Structure

    #delta-drawer (sidebar container)
  └─ #delta-drawer-panel (content panel)
      ├─ .delta-drawer-header (title + close button)
      │   ├─ #delta-table-search (search input)
      │   └─ #delta-table-search-btn (search button)
      └─ .delta-table-container (table wrapper)
          └─ [DeltaTableRenderer output]

    • Delta Table HTML Builder

      Pure functional module for constructing HTML table structures to display delta measurements between vertical line positions in COMTRADE waveform charts. This module generates complete table markup without performing DOM manipulation, following a functional programming approach for testability and reusability.

      Design Philosophy

      PrincipleDescription
      Pure FunctionsAll functions return HTML strings without side effects
      Separation of ConcernsHTML generation separate from DOM rendering
      Dynamic ColumnsTable structure adapts to number of vertical lines
      Theming SupportUses CSS variables for dark/light mode compatibility

      Key Features

      • Dynamic Header Generation — Creates columns for each vertical line value and delta pairs
      • Color-coded Columns — Uses crosshair colors to visually link values to chart cursors
      • Delta Calculations Display — Shows absolute delta and percentage change between consecutive lines
      • Channel Color Indicators — Color squares identify channels matching chart series
      • Time Row Support — Special handling for time reference row at top of table
      • Search/Filter Ready — Includes filterTableRows utility for channel filtering

      Table Structure

      | Channel | Line1 Val | Line2 Val | ... | Δ(1→2) | %(1→2) | Δ(2→3) | %(2→3) | ...
|---------|-----------|-----------|-----|--------|--------|--------|--------|
| Time    | T1        | T2        | ... | ΔT     | —      | ΔT     | —      |
| Ch1     | V1        | V2        | ... | ΔV     | %      | ΔV     | %      |
    See:
    • module:components/DeltaTableDataFormatter - Formats raw delta data for table consumption
    • module:components/DeltaTableRenderer - Renders generated HTML to DOM
    • module:components/DeltaDrawer - Parent drawer component
    Example
    // Build a complete delta table
import { buildTableHTML, filterTableRows } from './DeltaTable.js';

const tableData = [
  { channel: '__TIME_ROW__', v0: '0.00 μs', v1: '10.00 μs', delta0: '10.00 μs' },
  { channel: 'Voltage_A', color: '#ef4444', v0: '120.5', v1: '118.2', delta0: '-2.3', percentage0: -1.9 }
];

const html = buildTableHTML(tableData, 2, ['0.00 μs', '10.00 μs'], ['red', 'blue']);
container.innerHTML = html;

// Filter by channel name
const filtered = filterTableRows(tableData, 'Voltage');

    • Delta Table Data Formatter

      Pure transformation module that consolidates raw delta measurement data from multiple charts into a unified table format. Handles the complex task of merging channel data across different chart instances while maintaining proper column alignment for any number of vertical measurement lines.

      Design Philosophy

      PrincipleDescription
      Data ConsolidationMerges channel data from multiple charts into single rows
      Flexible Line CountSupports 1 to N vertical lines with proper delta pairing
      Pure TransformationNo side effects, returns new formatted data structure
      Graceful DegradationHandles missing data with N/A placeholders

      Key Features

      • Single Line Mode — When only 1 vertical line exists, shows values without delta columns
      • Multi-Line Mode — For 2+ lines, calculates deltas between consecutive pairs
      • Channel Deduplication — Same channel from different charts merged into single row
      • Time Row Generation — Automatically creates time reference row with timestamps
      • Delta Time Grouping — Groups sections by unique delta times for proper pairing
      • Missing Data Handling — Fills gaps with "N/A" for incomplete channel data

      Data Flow

      Input: deltaData[] from multiple charts
  ↓
Group by deltaTime (identifies line pairs)
  ↓
Build channelMap (merge same channels)
  ↓
Extract v0, v1, v2... (value at each line)
  ↓
Calculate delta0, delta1... (between pairs)
  ↓
Add percentage0, percentage1...
  ↓
Insert time row at position 0
  ↓
Output: tableData[] ready for DeltaTable
    See:
    • module:components/DeltaTable - Consumes formatted data for HTML generation
    • module:components/DeltaDrawer - Provides raw deltaData input
    • module:components/verticalLineControl - Source of vertical line positions
    Example
    // Format delta data from multiple charts
import { formatTableData } from './DeltaTableDataFormatter.js';

const deltaData = [
  {
    deltaTime: '10.00 μs',
    series: [
      { name: 'Voltage_A', color: '#ef4444', v1Formatted: '120.5', v2Formatted: '118.2', 
        deltaFormatted: '-2.3', percentage: -1.9 }
    ]
  }
];

const verticalLineTimes = ['0.00 μs', '10.00 μs'];
const tableData = formatTableData(deltaData, 2, verticalLineTimes);

// Result: [{ channel: '__TIME_ROW__', ... }, { channel: 'Voltage_A', v0: '120.5', ... }]

    • Delta Table DOM Renderer

      Reactive rendering component that manages the DOM lifecycle for delta measurement tables. Integrates with the application's reactive state system (createState) to automatically re-render tables when vertical line positions change, ensuring the displayed data always reflects the current cursor positions.

      Design Philosophy

      PrincipleDescription
      Reactive UpdatesSubscribes to state changes for automatic re-rendering
      Factory PatterncreateDeltaTableRenderer returns API object with methods
      Clean LifecycleProvides destroy() for proper cleanup of subscriptions
      State EncapsulationInternal state tracks current data and line count

      Key Features

      • Automatic Re-rendering — Subscribes to verticalLinesX state changes
      • Time Extraction — Converts numeric positions to formatted time strings
      • Flexible State Access — Handles multiple state API patterns (asArray, value, direct)
      • Fallback Placeholders — Generates T1, T2... labels when times unavailable
      • Debug Logging — Comprehensive console output for troubleshooting
      • Resource Cleanup — destroy() method unsubscribes and clears DOM

      Component Lifecycle

      Create: createDeltaTableRenderer(container, state)
   ↓
Auto-subscribe to verticalLinesX state
   ↓
render(tableData, lineCount) → DOM update
   ↓
State changes → auto re-render
   ↓
destroy() → unsubscribe + clear DOM
    See:
    • module:components/DeltaTable - Generates HTML consumed by this renderer
    • module:components/DeltaTableDataFormatter - Formats data before rendering
    • module:components/createState - Reactive state system for subscriptions
    • module:utils/constants - crosshairColors array for table styling
    Example
    // Create and use a delta table renderer
import { createDeltaTableRenderer } from './DeltaTableRenderer.js';
import { createState } from './createState.js';

const container = document.getElementById('delta-table-container');
const verticalLinesXState = createState([]);

const renderer = createDeltaTableRenderer(container, verticalLinesXState);

// Render table data
const tableData = formatTableData(deltaData, 2, times);
renderer.render(tableData, 2);

// Access current state
console.log(renderer.currentData);       // Current table data
console.log(renderer.currentLinesCount); // Number of lines

// Cleanup when done
renderer.destroy();

    Delta Table Data Formatter

    Pure transformation module that consolidates raw delta measurement data from multiple charts into a unified table format. Handles the complex task of merging channel data across different chart instances while maintaining proper column alignment for any number of vertical measurement lines.

    Design Philosophy

    PrincipleDescription
    Data ConsolidationMerges channel data from multiple charts into single rows
    Flexible Line CountSupports 1 to N vertical lines with proper delta pairing
    Pure TransformationNo side effects, returns new formatted data structure
    Graceful DegradationHandles missing data with N/A placeholders

    Key Features

    • Single Line Mode — When only 1 vertical line exists, shows values without delta columns
    • Multi-Line Mode — For 2+ lines, calculates deltas between consecutive pairs
    • Channel Deduplication — Same channel from different charts merged into single row
    • Time Row Generation — Automatically creates time reference row with timestamps
    • Delta Time Grouping — Groups sections by unique delta times for proper pairing
    • Missing Data Handling — Fills gaps with "N/A" for incomplete channel data

    Input: deltaData[] from multiple charts
  ↓
Group by deltaTime (identifies line pairs)
  ↓
Build channelMap (merge same channels)
  ↓
Extract v0, v1, v2... (value at each line)
  ↓
Calculate delta0, delta1... (between pairs)
  ↓
Add percentage0, percentage1...
  ↓
Insert time row at position 0
  ↓
Output: tableData[] ready for DeltaTable
